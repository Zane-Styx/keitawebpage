# Codebook

## Dataset: isp-dataset.csv

### Variables
- record_id: Unique record identifier
- isp: Internet service provider (PLDT, Globe, Converge, Meridian, Cablelink)
- download_mbps: Measured download speed in Mbps
- upload_mbps: Measured upload speed in Mbps
- ping_ms: Latency in milliseconds
- jitter_ms: Jitter in milliseconds
- peak_period: Peak or off-peak test window
- connection_type: Wi-Fi or Ethernet
- device_type: Device used during testing
- service_quality_score: SERVQUAL composite score
- user_satisfaction_score: Satisfaction score (Likert scale)

## Notes
- Data is anonymized and aggregated where needed.
- Replace with real values as data collection progresses.
